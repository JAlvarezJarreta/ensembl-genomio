Environment setup¶

1python3.7 -m venv path/to/virtual_env
2. path/to/virtual_env/bin/activate
3pip install -r requirements.txt -e .

Do not forget to load this environment by sourcing virtual_env/bin/activate in order to run ensembl-genomio pipeline(s).
