ensembl.io.genomio.schemas.json package¶

Handling and verification of JSON schemas module.

Submodules¶

ensembl.io.genomio.schemas.json.factory module¶

Generates one JSON file per metadata type inside manifest, including the manifest itself.

ensembl.io.genomio.schemas.json.factory.json_schema_factory(manifest_dir: PathLike, metadata_types: List[str], output_dir: PathLike) → None[source]¶

Generates one JSON file per metadata type inside manifest, including “manifest.json” itself.

Each JSON file will have the file name of the metadata type, e.g. “seq_region.json”.

Parameters:

  • manifest_dir – Path to the folder with the manifest JSON file to check.

  • metadata_types – Metadata types to extract from manifest as JSON files.

  • output_dir – Path to the folder where to generate the JSON files.

ensembl.io.genomio.schemas.json.validate module¶

Validates a JSON file with the provided JSON schema.

ensembl.io.genomio.schemas.json.validate.json_schema_validator(json_file: PathLike, json_schema: PathLike) → None[source]¶

Validates a JSON file with the provided JSON schema.

Parameters:

  • json_file – Path to the JSON file to check.

  • json_schema – JSON schema to validate json_file against.

Example

check_json_schema –json_file <JSON_FILE> –json_schema <JSON_SCHEMA>
