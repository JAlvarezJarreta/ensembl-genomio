ensembl.io.genomio.database package¶

Ensembl core database interface module.

Submodules¶

ensembl.io.genomio.database.factory module¶

Generates one JSON file per metadata type inside manifest, including the manifest itself.

Can be imported as a module and called as a script as well, with the same parameters and expected outcome.

ensembl.io.genomio.database.factory.format_db_data(server: CoreServer, dbs: List[str], brc_mode: bool = False) → List[Dict][source]¶

Returns metadata from a list of databases on a server.

Parameters:

  • server – Server where all the databases are hosted.

  • dbs – List of database names.

  • brc_mode – If true, assign BRC4.organism_abbrev as the species, and BRC4.component as the division. Otherwise, the species will be species.production_name and the division will be species.division.

Returns:

“database”, “species” and “division”.

Return type:

List of dictionaries with 3 keys
