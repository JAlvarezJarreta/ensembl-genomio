ensembl.io.genomio.genome_metadata.prepare.add_provider(genome_data: Dict, gff3_file: PathLike | None = None) → None[source]¶

Adds provider metadata for assembly and gene models in genome_data.

Assembly provider metadata will only be added if it is missing, i.e. neither provider_name or provider_url are present. The gene model metadata will only be added if gff3_file is provided.

Parameters:

  • genome_data – Genome information of assembly, accession and annotation.

  • gff3_file – Path to GFF3 file to use as annotation source for this genome.

ensembl.io.genomio.genome_metadata.prepare.add_species_metadata(genome_data: Dict, base_api_url: str = 'https://www.ebi.ac.uk/ena/browser/api/xml/') → None[source]¶

Adds missing species metadata based on the genome’s accession.

The taxonomy_id, strain and scientific_name will be fetched from the taxonomy information linked to the given accession.

Parameters:

  • genome_data – Genome information of assembly, accession and annotation.

  • base_api_url – Base API URL to fetch the taxonomy data from.

ensembl.io.genomio.genome_metadata.prepare.get_taxonomy_from_accession(accession: str, base_api_url: str = 'https://www.ebi.ac.uk/ena/browser/api/xml/') → Dict[source]¶

Returns the taxonomy metadata associated to the given accession.

Parameters:

  • accession – INSDC accession ID.

  • base_api_url – Base API URL to fetch the taxonomy data from.

Returns:

Dictionary with key-value pairs for taxon_id and scientific_name. strain will be added only if present in the fetched taxonomy data.

Raises:

MissinDataException – If TAXON_ID or SCIENTIFIC_NAME are missing in the taxonomy data fetched.

ensembl.io.genomio.genome_metadata.prepare.prepare_genome_metadata(input_file: PathLike, output_file: PathLike, gff3_file: PathLike | None = None, base_api_url: str = 'https://www.ebi.ac.uk/ena/browser/api/xml/') → None[source]¶

Updates the genome metadata JSON file with additional information.

In particular, more information is added about the provider, the assembly and its gene build version, and the taxonomy.

Parameters:

  • input_file – Path to JSON file with genome metadata.

  • output_file – Output directory where to generate the final genome.json file.

  • gff3_file – Path to GFF3 file to use as annotation source for this genome.

  • base_api_url – Base API URL to fetch the taxonomy data from.
