API Setup and installation¶

Requirements¶

An Ensembl API checkout including:

  • ensembl-genomio ensembl-genomio. (export /src/perl and /src/python into PERL5LIB and PYTHONPATH respectively)

  • ensembl-hive ensembl-hive.

  • ensembl-production ensembl-production.

  • ensembl-analysis ensembl-analysis. (on dev/hive_master branch)

  • ensembl-taxonomy ensembl-taxonomy.

  • ensembl-orm ensembl-orm.

Software¶

  1. Python 3.8+

  2. Bioperl 1.6.9+

Python Modules¶

  1. bcbio-gff

  2. biopython

  3. jsonschema

  4. mysql-connector-python

  5. requests

  6. python-redmine

Installation¶

Directly from GitHub:

1git clone https://github.com/Ensembl/ensembl-genomio
2git clone https://github.com/Ensembl/ensembl-analysis -b dev/hive_master
3git clone https://github.com/Ensembl/ensembl-production
4git clone https://github.com/Ensembl/ensembl-hive
5git clone https://github.com/Ensembl/ensembl-taxonomy
6git clone https://github.com/Ensembl/ensembl-orm
