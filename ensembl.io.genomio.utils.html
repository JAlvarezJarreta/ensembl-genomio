ensembl.io.genomio.utils.archive_utils module¶

Utils to deal with archive files.

ensembl.io.genomio.utils.archive_utils.extract_file(src_file: PathLike, dst_dir: PathLike) → None[source]¶

Extracts the src_file into dst_dir.

If the file is not an archive, it will be copied to dst_dir. dst_dir will be created if it does not exist.

Parameters:

  • src_file – Path to the file to unpack.

  • dst_dir – Path to where extract the file.

ensembl.io.genomio.utils.archive_utils.open_gz_file(file_path: PathLike) → Generator[TextIO, None, None][source]¶

Yields an open file object, even if the file is compressed with gzip.

The file is expected to contain a text, and this can be used with the usual “with”.

Parameters:

file_path – A file path to open.
